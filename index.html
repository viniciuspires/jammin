Jammin' Online

Open source Javascript music score editor.

View project onGitHub

What?

Jammin' Online is a web music score editor and player, maintained by a community of musician programmers, free for our fellow musician users.

Currently, the project is being developed, thanks to Google's Web Audio API suggestion to W3C, and should be soon in beta testing.

How?

The goal is to make a Javascript (actually, Coffescript) collaborative music composing web experience, tying together @mudcube's MIDI.js and @0xfe's VexFlow.

The file handled by the main project is MusicJSON (because it's JS). Next file type must be MusicXML (using @saebekassebil's MusicJSON to MusicXML Parser).

Also, because it's beautiful & ready, Twitter's Bootstrap is used for the application frontend.

Why?

Because music composition should be free and social. That's our dream.
